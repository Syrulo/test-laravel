## CRUD Bibliothèque (Laravel)

Petit projet personnel réalisé pour découvrir rapidement Laravel. Il m’a permis de me familiariser avec le framework, sa structure MVC, l’ORM Eloquent et la gestion des routes et vues. Il est perfectible et constitue une première approche pratique.

## Description

Application permettant de gérer une bibliothèque avec des livres, offrant les opérations CRUD (Créer, Lire, Mettre à jour, Supprimer).

## Fonctionnalités

- **Gestion des livres : création, modification, suppression et consultation**
- **Organisation simple des données pour un apprentissage rapide du framework**

## Technologies utilisées

- **PHP 8+** avec Laravel

- **MySQL** pour la base de données

- **Blade** pour les templates front-end

- **Tailwind CSS** pour le design simple et responsive
